SYARAT DAN KETENTUAN PENGGUNAAN

PERSETUJUAN PENGGUNAAN: Dengan menggunakan Tool ini, Anda menyetujui seluruh ketentuan berikut.

1. FUNGSI TOOL

Tool ini HANYA mengisi otomatis pilihan "Tidak", "Tidak sama sekali", "Normal" atau "Belum" pada formulir di browser Anda. Tool bekerja secara lokal dan memerlukan klik manual untuk diaktifkan.

2. APA YANG TIDAK DILAKUKAN TOOL

Tool ini TIDAK:

  • Mengirim data ke server manapun
  • Mengakses atau menyimpan data pribadi/pasien
  • Memodifikasi sistem aplikasi
  • Submit formulir secara otomatis

3. TANGGUNG JAWAB PENGGUNA

Anda WAJIB:

  • Memverifikasi ulang seluruh jawaban sebelum submit
  • Bertanggung jawab penuh atas keakuratan data
  • Mematuhi kebijakan aplikasi/website dan ketentuan hukum yang berlaku

4. BATASAN TANGGUNG JAWAB

Pengembang TIDAK bertanggung jawab atas:

  • Kesalahan pengisian data
  • Konsekuensi atau hukum dari penggunaan Tool
  • Gangguan atau error yang mungkin terjadi

5. PRIVASI

Tool ini TIDAK mengumpulkan, menyimpan, atau mengirim data apapun. Semua operasi dilakukan lokal di browser Anda.
